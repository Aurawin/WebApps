• DBMS - supported systems include MySQL, Postgres, and Oracle
  • Proprietary - with the exception of a dbms,  Aurawin does not require additional server licensing, virtualization software, or server managment
  • Highly Scalable - scale Aurawin with as many clusters & nodes as needed
  • Core Objects - provide security, features & back-end to apps on the platform.
  • Robust - No middleware, active server pages, java, or php scripting
  • Streamlined - restful back-end core objects
  • Indexed - one engine for all network protocols
  • Smart - hung processes automatically terminate & restart
  • Secure - Core objects have access controls, and OpenSSL provides transport layer security
