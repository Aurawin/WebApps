• Open Source - developers, businesses, consultants can fix, and improve the Aurawin VDM
  • Browser for device virtualization - works on all HTML5 fully compatible browsers
  • First-time boots in under 10 seconds - VDM boots when page loads
  • Reboots much faster - return loads much faster
  • Browser tweaks - commited support for FireFox, Safari, and Chrome
  • Smart - full featured apps on all devices via Navigation UI component
  • Private - Apps can run inside the VDM with authenticated users
  • Public - Apps can run as a single page with no login required
  • Public & Private - Private Apps can produce content for public Apps

The best advantage is that once an application is developed, it is available to anyone who has access to your cloud.
