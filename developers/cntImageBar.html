Pure Cloud Software

Aurawin Virtual Desktop Machine (AuVDM) is an open source project that leverages the power of CSS & JavaScript to emulate a vdm on any HTML5 capable device. 

Aurawin Social Computing Server (AuSCS) is prerequisite software for Linux, Apple, and Windows computers. 

Use AuVDM & AuSCS to create public or private cloud services, apps, and more.
